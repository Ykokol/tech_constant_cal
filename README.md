# 工程常数计算工具

## 项目概述

本项目是一个工程常数计算工具，主要用于工程实践过程中的常数计算，避免人工计算错误，提高工作效率和准确性。

## 核心功能

1. **EP阴极暴露长度计算**
   - 输入阴极宽度和样品参数（长、宽、厚、数量）
   - 支持添加多个样品
   - 自动计算阴极暴露长度

2. **样品快捷分割**
   - 输入样品尺寸需求
   - 输入边角料几何信息
   - 计算最多可切出的样品数量和切割顺序
   - 考虑线切割线径（0.42mm）

3. **开发者设置**
   - 管理配置文件
   - 修改默认参数（如阴极面积与样品面积的比例）
   - 配置保存和加载

## 技术实现

- **前端**：HTML、CSS、JavaScript
- **存储**：localStorage（本地存储）
- **响应式设计**：适配不同屏幕尺寸

## 使用方法

1. 打开 `index.html` 进入主界面
2. 选择需要使用的功能：
   - 点击「EP阴极暴露长度计算」进入相应页面
   - 点击「样品快捷分割」进入相应页面
   - 点击「开发者设置」进入相应页面
3. 按照页面提示输入参数并点击计算按钮
4. 查看计算结果

## 注意事项

- 本应用使用本地存储保存配置和样品信息，刷新页面后数据不会丢失
- 请确保输入有效的数值参数，否则计算结果可能不准确
- 样品分割功能目前仅支持矩形边角料的切割计算
